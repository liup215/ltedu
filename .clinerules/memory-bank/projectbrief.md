# Project Brief: A-level Toolkit

## 1. Overview

A-level Toolkit is a comprehensive online education platform with a modular, function-oriented architecture. The system is designed to deliver a rich learning experience for students and robust management tools for educators and administrators. The platform consists of a Go backend API and a Vue.js/TypeScript frontend web application.

## 2. Functional Modules

The project is organized around the following core website functions:

1. **User & Access Management**
   - Registration, login, profile management, and role-based access (Student, Teacher, Admin).

2. **System Administration**
   - Admin dashboard, user management, system settings, teacher application review.

3. **Educational Structure Management**
   - Organisation, qualification, syllabus, and chapter management.

4. **Course Management**
   - Course creation, editing, and video content management.

5. **Examination & Assessment**
   - Question bank, exam paper builder, random/past paper management, paper series/code management.

6. **Content & Media Management**
   - Document, image, video, and slide management.

7. **AI & Advanced Features**
   - Vocabulary sets, AI-powered features (e.g., question generation, recommendations).

## 3. Key Objectives

- Build a scalable, maintainable, and extensible e-learning platform.
- Provide seamless user experiences for all roles.
- Support diverse educational content and assessment workflows.
- Build the **Syllabus Navigator (Learning Engine)**: a syllabus-centric, closed-loop learning engine built on **Syllabus/Chapter/Question/Paper** (Goal → Diagnostic → Task Stream → Attempt logging → SRS → Sprint), independent of entry points (Web/App/MCP).
- Ensure secure user management and data persistence.
- Integrate cloud services for media and AI capabilities.
